# Experiment 1 – Network Topologies

## Files Included
- exp1_topologies.pkt
- output_exp1.txt
- report_exp1.pdf
- README.md

## How to Test
1. Open Cisco Packet Tracer.
2. Load exp1_topologies.pkt file.
3. Use Simulation Mode to observe packet flow.
4. Perform ping tests between PCs.
5. Disconnect one switch link in ring topology to test fault tolerance.

## Description
This experiment demonstrates Star, Hub-based and Ring-like topologies and compares their connectivity, performance and fault tolerance.
